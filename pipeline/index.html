All the other Curlers didn't work! So I wrote my own, from the docs

INFO

I used to use a particular PHP Multi-Curl library, and it was doing the job for the most part. Enabling me to do hundreds of calls per minute.

My particular use case is to get allot of content, simultaneously. With one twist, I needed to be able to pass context along with the request. One the response comes back, I can then process the content based on the context.
